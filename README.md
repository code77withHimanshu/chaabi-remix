## Clone the Repository

Clone the repository using the following command:

```shellscript
git clone git@github.com:code77withHimanshu/chaabi-remix.git
```

## Install Dependencies

Open a new terminal and run the following command:

```shellscript
npm install
```

## Development

Run the dev server:

```shellscript
npm run dev
```
